Soldier's Memory & Attention game

0 Points

1. You get one sentence (then two, three etc).

2. The countdown starts and you should try to remember the sentence

3. When countdown ends, the sentence will be hidden and you'll see an input where You will write a sentence you remembered.

4. By clicking on submit button You'll see this sentence again.

5. Check how many mistakes you made, write that amount into 'mistakes field'.

6. You'll get more points for less mistakes - points are shown at the top-left corner.

Have fun and focus!
